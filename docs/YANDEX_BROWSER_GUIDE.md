# Yandex Browser — совместимость

## Известные ограничения и решения

### Сайд-панель не работает
**Проблема**: Сайд-панель расширения в Яндекс.Браузере недоступна.

**Решение**: Используйте **popup** — нажмите на иконку расширения в панели инструментов. Чат и все функции работают так же, как в сайд-панели.

### История чата не сохраняется
**Проблема**: История может не сохраняться между перезагрузками страницы.

**Причина**: IndexedDB в Яндекс.Браузере (или в режиме инкогнито) может быть ограничен.

**Что делать**: Используйте обычный режим браузера или Chrome для полного сохранения истории. В текущей сессии история сохраняется.

## Поддержка по браузерам

| Функция | Chrome | Yandex (popup) |
|--------|--------|----------------|
| Чат | ✅ | ✅ |
| История сообщений | ✅ | ⚠️ только сессия |
| Markdown, источники, spaces | ✅ | ✅ |
| Настройки LLM | ✅ | ✅ |

**Chrome** — полная поддержка (сайд-панель + IndexedDB).  
**Yandex / Edge** — только popup, без сайд-панели.

---

## Установка в Яндекс.Браузере

1. Установите расширение (Chrome Web Store или Load unpacked).
2. В настройках браузера: Расширения → включите расширение.
3. Нажимайте на иконку расширения для открытия чата (popup).

Подробнее о принципах расширения — в [README](../README.md).
